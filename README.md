## 📖 Overview

**pandey** is a streamlined, lightning-fast web application built to serve as a high-fidelity digital portal. Designed with standard-compliant frontend practices, the project prioritizes raw speed, high SEO accessibility, and responsiveness.

Configured with a custom domain pointer (`CNAME`) and deployable straight to GitHub Pages, this repository offers an elegant template for immediate, lightweight web publishing.

- ⚡ **Zero-Latency Performance:** Pure semantic structure ensuring sub-second loading speeds.

## 📂 Project Structure

A clean, predictable, and modular structural layout:

```bash
pandey/
├── CNAME          # Custom domain mapping configuration
├── README.md      # Project documentation (you are here)
└── index.html     # Main application entry point and structural markup
```

## 🚀 Installation & Setup

Set up a local development copy in under 60 seconds.

### Prerequisites

You only need a modern web browser and a code editor (like VS Code).

### Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Alok345/pandey.git
   ```

2. **Navigate into the project directory:**
   ```bash
   cd pandey
   ```

3. **Launch the project:**
   * Double-click `index.html` to open it in your default web browser.
   * *Alternatively*, use the VS Code [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension to run a local hot-reloading development server:
     ```bash
     # If using local http-server via npm
     npx http-server .
     ```

## 🔧 Deployment Configuration

### GitHub Pages Setup

1. Push your repository to your GitHub account.
2. Navigate to **Settings > Pages** on your GitHub repository page.
3. Under **Build and deployment**, set the Source to **Deploy from a branch**.
4. Choose the `main` (or `master`) branch and directory root (`/`), then click **Save**.

### Setting Up Custom Domain (`CNAME`)

1. Open the `CNAME` file.
2. Replace its contents with your own custom domain:
   ```txt
   yourdomain.com
   ```
3. Update your domain registrar's DNS settings by adding a `CNAME` record pointing to `Alok345.github.io`.
